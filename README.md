# pizza_sales_project

---

## 📌 Overview

This project focuses on analyzing **pizza sales data** using **Microsoft Excel and SQL**. It involves working with multiple datasets to understand order patterns, revenue generation, customer preferences, and product performance.

The analysis helps uncover key insights such as **top-selling pizzas, peak order times, and revenue contribution**, enabling better business decisions.

---

## 🎯 Objective

* To analyze total orders and revenue generated  
* To identify best-selling pizza types and categories  
* To understand customer ordering patterns  
* To analyze sales distribution by time (hour/day)  
* To evaluate product pricing and performance  
* To calculate revenue contribution by each pizza type  

---

## 📂 Dataset

The dataset consists of four CSV files:

* **order_details.csv**  
* **orders.csv**  
* **pizza_types.csv**  
* **pizzas.csv**  
 
---

## 🛠️ Tools & Technologies

* **Microsoft Excel**

  * Data Creation & CSV Management  

* **MySQL**

  * Data Analysis  
  * Advanced SQL Queries  

---

## 🔍 Steps Performed

* Created structured datasets in Excel (CSV format)  
* Imported data into MySQL database  
* Established relationships between tables  
* Performed SQL queries using JOIN, GROUP BY, ORDER BY  
* Conducted revenue and sales analysis  
* Extracted time-based and category-based insights  

---

## 📊 SQL Analysis & Insights

### 📦 Order & Revenue Analysis

* Calculated total number of orders placed  
* Computed total revenue generated from pizza sales  
* Analyzed cumulative revenue over time  

---

### 🍕 Product Analysis

* Identified the **highest-priced pizza**  
* Found the **most common pizza size ordered**  
* Listed **top 5 most ordered pizza types**  
* Determined **top 3 pizzas based on revenue**  

---

### 📊 Category Analysis

* Calculated total quantity ordered by each pizza category  
* Identified top-performing categories  
* Found top 3 pizzas by revenue within each category  

---

### ⏰ Time-Based Analysis

* Analyzed distribution of orders by **hour of the day**  
* Grouped orders by date to calculate **average pizzas ordered per day**  

---

### 💡 Revenue Contribution

* Calculated percentage contribution of each pizza type to total revenue  
* Identified high-revenue generating products  

---

## 📈 Key Insights

* A few pizza types contribute the majority of total revenue  
* Certain pizza sizes (like Medium/Large) are ordered more frequently  
* Peak order hours indicate high customer activity periods  
* Top-selling pizzas vary across categories  
* Revenue trends help identify business growth patterns  
* Category-wise analysis helps optimize menu offerings  

---

## 🚀 Conclusion

This project demonstrates how **Excel and SQL together** can be used to perform comprehensive data analysis. It highlights key business insights related to **sales performance, customer behavior, and revenue trends** in a pizza store.
